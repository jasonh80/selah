# Selah

Your daily Bible chapter, made visual, simple, and personal.
Learn more. Dive deeper. Grow closer to Jesus.

Phase 0 — static prototype (Exodus 27). Next.js + React + Tailwind + TypeScript.

## Run locally
```
npm install
npm run dev
```
Open http://localhost:3000

## Deploy (Netlify)
Push to GitHub → Netlify → Import from GitHub → Deploy. Next.js is auto-detected; `netlify.toml` is included.
